HTML Tag Usage

I used almost all the HTML tags we talked about in the lesson. In the head part, I used to describe how the website should look on different devices and added some extra details like a description. For connecting JavaScript files, I used "script" tags, and for style sheets, I used "link" tags. All the other tags were used inside the body. The most common ones were: "nav" for navigation, "div", "a", "button", "ul", "li", "header" for headings, "section" to create containers, "p" for paragraphs, "h1" and "h2" for different heading levels, and "footer"

Bootstrap Components Used

I closely followed the Bootstrap documentation and applied various components to enhance my website. One key feature I incorporated is the navigation bar (navbar). I designed it with a specific class for styling: 'navbar navbar-expand-lg navbar-light fixed-top.' Additionally, I utilized Bootstrap's built-in navigation toggle for creating a collapsible menu button. For the overall layout and styling, I used Bootstrap components like containers with padding, flexible properties, height adjustments, and alignment settings. This ensures a neat and responsive design throughout the website.

How I used media queries

I used almost all of boostraps in-build componets to make my website responsive, so I did not have to manually add media queries. But I styled the main image by specifying the dimensions of the bg-masthead image by giving the range of min-width and max-width at different screen sizes.

Gestalt principle that I chose

The Gestalt principle I have used here is "Promixity". I have placed likely componets together as a bunch so they go together. All the projects in the homepage are as a bunch so viewers are able to distinguish.
